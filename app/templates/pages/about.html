About:

Project Description:

GISalmon is an interactive web application used to monitor the health of British Columbia's various salmon populations. GISalmon was created as a final project for CSc 485D: Information Visualization by Connor Schultz, Ben Wilmot and Wyatt Timmermans.

The application is built on the Flask micro web-framework for python3 and deployed using Docker and Heroku. The overlays shown in the GISalmon Map were created using the Folium package for visualization and GeoPandas for data management. The animated line graphs shown in the GISalmon Time Series were created using matplotlib and pandas for python3.

Terminology:

Conservation Unit (CU): The Government of Canada defines a Conservation Unit or CU as a group of wild salmon that is sufficiently isolated from other groups that if it was to be made locally extinct, it would be very unlikely for the group to recover naturally within an acceptable time frame.

Wild Salmon Policy (WSP) Status: The Government of Canada defines Wild Salmon Policy Status or WSP Status as an evaluation of the biological status of a CU. The status is defined by a number of metrics which are integrated and expertly reviewed for each CU. A WSP Status of GREEN indicates a healthy status CU, an AMBER status indicates a marginal status CU and a RED status indicates a poor status CU. A WSP Status of DD represents a data-deficient CU and a TBD Status represents a CU under consideration.

More information for how WSP Status is defined can be seen at the Fisheries and Oceans Canada webpage.
